Как использовать

Использование API ноды
-----------------------

Блокчейн-платформа Восток предоставляет возможность взаимодействия клиентских приложений с блокчейн-платформой, как в части получения данных (транзакции, блоки, балансы и др.), так и в части записи инфомрации в блокчейн (подписание и отправка транзакций). Указанное взаимодействие реализуется посредством RESTful API ноды.

Правила формирования запросов к ноде приведены в разделе :ref:`rest-api-node`.
